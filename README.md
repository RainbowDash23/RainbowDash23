##  Proyectos destacados

###  [Realtime Chat API](https://github.com/RainbowDash23/realtime-chat-api)
API de mensajería y notificaciones en tiempo real con **Node.js, Express y Socket.io**. Incluye autenticación JWT, chats directos y grupales, indicadores de "escribiendo...", presencia online/offline, edición/borrado de mensajes, y tests automatizados con Jest.
`Node.js` `Express` `Socket.io` `PostgreSQL` `JWT` `Jest`

###  [Ecommerce API](https://github.com/RainbowDash23/ecommerce-api)
API REST para un e-commerce construida con **Django REST Framework**, con gestión de productos, carritos y órdenes sobre PostgreSQL.
`Python` `Django REST Framework` `PostgreSQL`

###  [Task Manager API](https://github.com/RainbowDash23/task-manager-api)
API REST de gestión de tareas con **Node.js y Express**, autenticación JWT y persistencia en PostgreSQL.
`Node.js` `Express` `PostgreSQL` `JWT`

###  [Little Lemon — Restaurant API](https://github.com/RainbowDash23/capstone-project)
API REST construida con **Django REST Framework** para la gestión de menú y reservas de un restaurante. Proyecto capstone del Meta Back-End Developer Certificate.
`Python` `Django` `DRF` `SQLite`

###  [SISSU — Módulo de Evaluación Nutricional (IMC)](https://github.com/RainbowDash23/Proyecto-IMC)
Aplicación web con **Flask** que calcula el IMC, clasifica resultados según la OMS y genera recomendaciones personalizadas mediante la **API de Claude (Anthropic)**. Incluye historial con gráficas interactivas.
`Python` `Flask` `SQLite` `Claude API` `Chart.js`

###  [Genshin Quest Guide](https://github.com/RainbowDash23/genshin-impact-guide)
Guía interactiva de misiones de mundo para las 6 regiones de Genshin Impact. Sistema de progreso con persistencia en localStorage, búsqueda y filtros.
`React` `TypeScript` `Tailwind CSS` `Vite`
